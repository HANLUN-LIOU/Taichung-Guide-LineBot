# 台中美食景點推薦LineBot

這是一個使用 Flask 和 LINE Messaging API 開發的台中旅遊推薦 Bot。用戶可以根據自己的需求查詢台中的景點、美食和其他服務。

## 功能

- 提供台中各區的美食和景點推薦
- 附上店家電話、地址、營業時間、評分以及Google地圖
- 讓用戶可以為推薦的項目進行評分
- 提供即時的天氣資訊
- 包含互動式的台中知識王遊戲
